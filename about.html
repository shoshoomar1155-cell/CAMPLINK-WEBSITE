About CampLink

CampLink is a demo website for student trips, created as a multi-page project using HTML, CSS and JavaScript.

Who is this website for?

CampLink is designed for school and university students who want to book safe and organized trips such as camping, hiking, city visits and cultural tours. The website guides the student from creating an account, to completing the profile, choosing a destination and finally confirming the booking.

Project details

This project is built as a classic multi-page website, similar to booking websites.

  • Pages: Home, Sign Up, Profile, Destinations, Destination Details, Events, Booking, Booking Success, About, Contact.
  • Navigation: each page reloads completely when moving between links.
  • Design style: camping colors (green tones) with simple cards and images.
  • Content language: English, and it can be translated to Arabic by changing the text.

Technologies used

  • HTML – basic structure, forms and tables (from course Chapter 1–3).
  • CSS – colors, layout, box model and simple responsive ideas (from Chapter 4–5).
  • JavaScript – reading values, simple validation and dynamic text (from Chapter 6).

All features in this project are created only using the techniques explained in the course chapters.
